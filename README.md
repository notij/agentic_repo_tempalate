## Agent 协作建议（Vibe Coding）

如果你在进行 Vibe Coding，这里有两个可供 Agent 使用的协作文件：

- `agent_log/agent_read.md`: agent所概括的整个项目内容
- `agent_log/agent_update.md`: agent更新read与commit前更新worklog的规则

可直接复制给其他 Agent 的提示词：
```text
阅读 `agent_log/agent_read.md`，了解这个项目。
```

```text
阅读 `agent_log/agent_update.md`，按规则更新。
```
